About ReconJT

ReconJT is a passive OSINT reconnaissance web interface built with Flask. It is designed to integrate with your existing Linux engine (reconjt.sh) via subprocess.run([...], capture_output=True, text=True) on Kali Linux.

What this UI provides

  • Responsive, dark-theme dashboard
  • Domain input and scan workflow
  • WHOIS / NSLOOKUP / DIG results cards
  • Copy and download (TXT + Print-to-PDF)
  • Loading spinner, error page, footer
